SSL :

choco install openssl -> localhost.cnf

[req]
distinguished_name = dn
x509_extensions = EXT
prompt = no

[dn]
CN = localhost

[EXT]
subjectAltName = DNS:localhost
keyUsage = digitalSignature
extendedKeyUsage = serverAuth

openssl req -x509 -out server.crt -keyout server.key ^
  -newkey rsa:2048 -nodes -sha256 -days 365 ^
  -config localhost.cnf -extensions EXT

(Double Click on server.crt and install the certificate in Trusted Root Certification Authorities)

see Appendix for cross platform instructions.



Appendix:

## Local HTTPS Certificates (Cross-Platform)

### Windows
See Example 02 walkthrough.

### Linux (Ubuntu)
1. Install mkcert + libnss3-tools
2. Run `mkcert -install`
3. Generate certs: `mkcert localhost 127.0.0.1 ::1`

### macOS
1. Install mkcert via Homebrew
2. Run `mkcert -install`
3. Generate certs: `mkcert localhost 127.0.0.1 ::1`
4. Verify trust in Keychain Access
